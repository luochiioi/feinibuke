# P5.3 Points Wallet And UX Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn reward records into a clear points wallet, keep task points auto-issued, and make the App/Admin views show one consistent points balance.

**Architecture:** Keep `rewards` as the source of truth for earned rewards. Add pure helpers that calculate points balance and ledger rows from normalized reward records: task rewards count as issued at `earnedAt`, route rewards count after `rewardClaimed === true`. App reads the same summary through `marker-center`; admin reads cross-user summaries through `admin-center`.

**Tech Stack:** uni-app x / UTS 5.07, uniCloud cloud objects, Node `node:test`, uni-admin Vue.

### Task 0: Fix Focus Navigation Runtime Crash

**Problem:** Real-device feedback showed `java.lang.ClassCastException: UniNormalPageImpl cannot be cast to UTSJSONObject` at `stores/useMapStore.uts` inside `returnToIndexForFocus()`. The current helper scans `getCurrentPages()` and casts page instances to `UTSJSONObject`; this is invalid in UTS 5.07.

**Target behavior:** Keep the focus payload flow, but remove page-object introspection. Callers must pass an explicit `deltaToIndex` based on their known navigation depth. If the caller cannot know the depth, use `reLaunch` fallback.

- `pages/my-checkins/my-checkins.uvue`: opened from homepage action sheet, use delta `1`.
- `pages/tasks/tasks.uvue`: opened from homepage task button, use delta `1`.
- `pages/task-detail/task-detail.uvue`: opened from tasks page, use delta `2`; if future direct entry is added, pass `0` to reLaunch.
- `pages/route-detail/route-detail.uvue`: opened from routes page, use delta `2`; if future direct entry is added, pass `0` to reLaunch.

- [x] **Step 1: Write failing source guard test**

Modify `utils/map-focus-navigation.test.js`:

```js
test('focus navigation does not cast getCurrentPages page instances', () => {
  const source = read('stores/useMapStore.uts')
  assert.equal(source.includes('as UTSJSONObject'), false)
  assert.equal(source.includes('page["route"]'), false)
  assert.match(source, /returnToIndexForFocus\(deltaToIndex: number\)/)
})
```

Run:


- [x] **Step 2: Change helper signature**

Update `stores/useMapStore.uts`:

```ts
export function returnToIndexForFocus(deltaToIndex: number): void {
  if (deltaToIndex > 0) {
    uni.navigateBack({ delta: deltaToIndex })
    return
  }
  uni.reLaunch({ url: '/pages/index/index' })
}
```

Do not call `getCurrentPages()` inside this helper. Do not cast page objects.
